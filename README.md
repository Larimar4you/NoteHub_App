# NoteHub App

NoteHub is a web application for managing personal notes.
It allows users to create, search, view, and delete notes
with a clean interface and efficient data handling.

## Live Demo

note-hub-app-fawn.vercel.app

## Repository

https://github.com/Larimar4you/05-notehub

---

## Features

- Search notes with debounced input
- Paginated list of notes
- Create notes using a modal form
- Delete notes with automatic data refresh
- Form validation with clear error messages
- Server state management with caching
